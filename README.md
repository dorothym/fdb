# Readme.md for Stackathon project "Audio Repo"

## Technologies and libraries used:

* FSG
* Angular NG Audio
* Angular Filepicker
* Filestack
* Amazon S3
* Placehold.it

### Credits
* Theme base: startbootstrap.com
* Stock photos: pexels.com

## What is working: 

* Upload a file to S3
* File tags are saved to DB on upload
* Browse for a file
* About page
* Navbar
* File detail page
  ** audio player
  ** audio file download
